Інформаційна система створення та розсилки запрошень

Information system for creating and sending invitations

Опис ✍

Description ✍

📎 На сьогоднішній день існують численні онлайн-сервіси для створення креативних запрошень, проте більшість із них зосереджені виключно на візуальному оформленні. Сучасний ринок потребує більш функціональних рішень, які не лише допомагають у створенні естетично привабливих запрошень, а й забезпечують ефективний процес їх розсилки та управління. Проєкт "Invitations" спрямований на забезпечення користувачів усіма необхідними функціями для спрощення та пришвидшення процесу організації свят.

Keywords 📌

📎 Technology, Invitations, Events, Celebrating, Friends

Мета 🎯

Mission 🎯

📎 Метою проєкту є розроблення інформаційної системи для полегшення та пришвидшення створення простих і зрозумілих запрошень на будь-які події.

Основні завдання 📃

Main Tasks 📃

📎 Розроблений інструмент надасть користувачам можливість запросити друзів на свято лише заповнивши коротеньку форму та додавши їх до списку запрошених.

Завдання для реалізації функціоналу додатку:

  1. Реєстрація користувачів
  2. Авторизація користувачів
  3. Форма для вияснення інформації про подію
  4. Форма для зміни інформації про подію
  5. Можливість змінювати аватар у кабінеті користувача
  6. Відправка текстового запрошення до обраних друзів

Очікувані результати 📍

Expected Results 📍

📎 Результатом буде готовий веб-додаток який допоможе користувачам без додаткових роздумів заповнити всю необхідну інформацію про майбутнє свято, зберегти цю інформацію з можливістю змін та автоматичній розсилці запрошень на пошту обраних користувачів.
